## 💻 Set Up

### ⚗️ Virtual environment

Creating and activating the virtual environment:

```console
$ python3 -m venv .venv
$ source .venv/bin/activate
```

### 🦄 Django

Installing the latest version of Django (tested with Django 5.0b1):

```console
$ python3 -m pip install -r requirements.txt
```
